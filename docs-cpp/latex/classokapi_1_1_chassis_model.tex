\doxysubsection{Detailed Description}
A version of the \mbox{\hyperlink{classokapi_1_1_read_only_chassis_model}{Read\+Only\+Chassis\+Model}} that also supports write methods, such as setting motor speed. Because this class can write to motors, there can only be one owner and as such copying is disabled.
