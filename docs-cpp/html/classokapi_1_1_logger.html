static void okapi::Logger::setLogLevel ( LogLevel  level)
staticnoexcept

Set a new logging level. Log statements above this level will be disabled. For example, if the level is set to LogLevel::warn, then LogLevel::warn and LogLevel::error will be enabled, but LogLevel::info and LogLevel::debug will be disabled.
