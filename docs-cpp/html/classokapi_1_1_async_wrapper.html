A wrapper class that transforms an IterativeController into an AsyncController by running it in another task. The input controller will act like an AsyncController.

Parameters
iinputcontroller input, passed to the IterativeController
ioutputcontroller output, written to from the IterativeController
icontrollerthe controller to use
irateSupplierused for rates used in the main loop and in waitUntilSettled
isettledUtilused in waitUntilSettled
iscalethe scale applied to the controller output
