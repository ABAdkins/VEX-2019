A gyroscope on the given ADI port. If the port has not previously been configured as a gyro, then the constructor will block for 1 second for calibration. The gyro measures in tenths of a degree, so there are 3600 measurement points per revolution.

Parameters
iportthe ADI port number
imultipliera value multiplied by the gyro heading value
